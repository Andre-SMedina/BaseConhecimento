Instalar Angular CLI

npm i -g @angular/cli

Criar projeto

gs new "nome do projeto"

Inicar servidor

ng serve

Criar um componente

ng generate component components/first-component

Importar um componente

No arquivo app.component.html na pasta app, abra uma tag e digite <app-"nome-do-componente"></app-"nome-do-componente">

Adicionar dados do componente ao template HTML

Vá no arquivo "nomeComponent.component.ts" e adicione as variáveis/propriedades ou funções na classe que está sendo exportada. Exemplo: 
export class nomeComponent {
              name: string = "Andre";
              age: number = 30;
              hobbies = ['Nadar', 'Correr', 'Ler'];
              car = { name: 'Polo', year: 2019 };
            }

Depois vá no arquivo "nomeComponent.component.html" e adicione o valor das variáveis utilizando {{variável}} Exemplos:

{{ name }}

{{ hobbies[1] }}

{{ car.name }}

Adicionar estilos/CSS estáticos

Global: Na pasta src, abra o arquivo styles.css e faça suas alterações

No componente: Na pasta src/app, abra o arquivo nomeComponent.component.css e faça suas alterações

Adiconar estilos/CSS dinâmicos

No arquivo nomeComponent.component.ts, adicione na classe a variável/propriedade: 

export class nomeComponent {
            color = 'yellow';
            size = 30 + 'px';
          }

No arquivo nomeComponente.component.html, adicione a seguinte linha:

<p [ngStyle]="{'font-size': size, color: color}">Testanto CSS dinâmico</p>

Adicionar classes dinâmicas

No arquivo nomeComponent.component.ts, adicione na classe a variável/propriedade: 

export class nomeComponent {
            classes = ['small-title', 'green-title'];
            underline = 'underline-title';
          }

No arquivo nomeComponente.component.html, adicione a seguinte linha: 

<p [ngClass]="classes">Testanto classes</p>
         <p [ngClass]="underline">Testando classes</p>

Compartilhar dados do componente pai para o componente filho

Para que um coponente filho tenha acesso a um dado do componente pai, primeiro criamos o dado no componente pai: 

export class AppComponent {
            userData = { email: 'pedro@gmail.com', role: 'admin' };
          }

No componente filho, importamos "input" do core do Angular import { Component, Input } from '@angular/core'; Depois adicionamos à classe da seguinte forma: 

export class NomeComponenteComponent {
            @Input() userData!: { email: string; role: string };
          }

Não é obrigatório usar o mesmo nome "userData"

No arquivo HTML do componente pai, passamos os dados da seguinte forma:

<app-nomeComponente [userData]="userData"></app-nomeComponente>

Por fim, no arquivo HTML do componente filho, utilizamos o dado da seguinte forma:

<p>Email: {{ userData.email }}</p>

<p>Função: {{ userData.role }}</p>

Utilizando if para renderizar algo no template

No arquivo nomeComponent.component.ts, adicione na classe a variável/propriedade: 

export class nomeComponent {
            canShow: boolean = true;
          }

No arquivo nomeComponente.component.html, adicione a seguinte linha: 

<p *ngIf="canShow">Esse texto pode ser exibido</p>

Caso a variável/propriedade seja falsa, o texto não será exibido

Criando eventos ao interagir com a página

No arquivo nomeComponent.component.ts, adicione na classe a variável/propriedade: 

export class nomeComponent {
            show: boolean = true;
            sowMessage(): void {
              this.show = !this.show
            }
          }

No arquivo nomeComponente.component.html, adicione a seguinte linha: 

<p *ngIf="show">Exibindo mensagem!</p>
            <button (click)="showMessage()">Cique em mim!</button>

Criando interfaces

As inerfaces são utilizadas para tornar o projeto mais padronizado, pois quando for utilizar a interface, deve seguir todo o modelo dela.

Para criar interfaces, vá na pasta app e nela crie o Arquivo.ts(Sempre com lettra maiúscula) com o nome da interface. Dentro do arquivo deve ficar assim: 

export interface Animal {
            name: string;
            age: number;
          }
Quando quiser usar a interface, deve-se importá-la no arquivo utilizando: import { Animal } from 'src/app/Animal'; 
animals: Animal[ ] = [
            { name: 'Turca', type: 'dog', age: 5 },
            { name: 'Tom', type: 'Cat', age: 7 },
          ];
animal: Animal = {name: "bob", type: "dog", age: 4} 
showAge(animal: Animal): void {
            this.animalDetails = `O pet ${animal.name} tem ${animal.age} anos!`;
          }

Alterando strings com pipe operator

É possível alterar strings diretamente no template através de pipes. No arquivo HTML faça assim:

<p>{{ "testando uppercase" | uppercase}}</p> torna todas as letras maiúsculas

<p>{{ "testando titlecase" | titlecase}}</p> torna as primeiras letras de cada palava maiúsculas

Espelhando o que é digitado no input

No arquivo "app.module.ts", importe o módulo "FormsModule" assim: import { FormsModule } from '@angular/forms';

Na linha onde está o "@NgModule({...})", dentro do array "imports", inclua o "FormsModule" assim: imports: [BrowserModule, FormsModule],

No arquivo HTML, utilize o "ngModel" assim:

<form>
            <input
              type="text"
              [(ngModel)]="name"
              name="name"
              placeholder="Digite seu nome"
            />
            <input type="submit" value="Enviar" />
      </form>
    <p>{{name}}</p>

No componente relacionado ao arquivo HTML onde foi utilizado o "ngModel", crie a variável/propriedade dentro da classe: "name: string = ' '; "

Utilizando Services

"Services" seria como um conjunto de funções que podem ser importadas em qualquer componente.

  1. Para criar um, vá no terminal e digite: ng generate service services/nomeDoService
  2. No componente onde deseja utilizar, importe o service, import { nomeDoService } from 'src/app/services/nomeDoService.service';, passe como parâmetro no "constructor", constructor(private service: nomeDoService) { }
  3. Vá no arquivo "nomeDoService.service.ts" e crie a função do service: 
    soma(num1: number, num2: number) {
                return num1 + num2  
              }
  4. No componente, crie a função que utilizará o service:
    5. somarNumeros(num1: number, num2: number): void {
              console.log(this.service.soma(num1, num2));
            }
