Vídeos

Curso Hora de Codar

Instalar Angular CLI

npm i -g @angular/cli

Criar projeto

gs new "nome do projeto"

Inicar servidor

ng serve

Criar um componente

ng generate component components/first-component

Importar um componente

No arquivo app.component.html na pasta app, abra uma tag e digite <app-"nome-do-componente"></app-"nome-do-componente">

Adicionar dados do componente ao template HTML

Vá no arquivo "nomeComponent.component.ts" e adicione as variáveis/propriedades ou funções na classe que está sendo exportada. Exemplo: 

export class nomeComponent {
              name: string = "Andre";
              age: number = 30;
              hobbies = ['Nadar', 'Correr', 'Ler'];
              car = { name: 'Polo', year: 2019 };
            }

Depois vá no arquivo "nomeComponent.component.html" e adicione o valor das variáveis utilizando {{variável}} Exemplos:

{{ name }}

{{ hobbies[1] }}

{{ car.name }}

Adicionar estilos/CSS estáticos

Global: Na pasta src, abra o arquivo styles.css e faça suas alterações

No componente: Na pasta src/app, abra o arquivo nomeComponent.component.css e faça suas alterações

Adiconar estilos/CSS dinâmicos

No arquivo nomeComponent.component.ts, adicione na classe a variável/propriedade:

export class nomeComponent {
            color = 'yellow';
            size = 30 + 'px';
          }

No arquivo nomeComponente.component.html, adicione a seguinte linha:

<p [ngStyle]="{'font-size': size, color: color}">Testanto CSS dinâmico</p>

Adicionar classes dinâmicas

No arquivo nomeComponent.component.ts, adicione na classe a variável/propriedade:

export class nomeComponent {
            classes = ['small-title', 'green-title'];
            underline = 'underline-title';
          }

No arquivo nomeComponente.component.html, adicione a seguinte linha:

<p [ngClass]="classes">Testanto classes</p>
         <p [ngClass]="underline">Testando classes</p>

Compartilhar dados do componente pai para o componente filho

Para que um coponente filho tenha acesso a um dado do componente pai, primeiro criamos o dado no componente pai:

export class AppComponent {
            userData = { email: 'pedro@gmail.com', role: 'admin' };
          }

No componente filho, importamos "input" do core do Angular import { Component, Input } from '@angular/core'; Depois adicionamos à classe da seguinte forma: 

export class NomeComponenteComponent {
            @Input() userData!: { email: string; role: string };
          }

Não é obrigatório usar o mesmo nome "userData"

No arquivo HTML do componente pai, passamos os dados da seguinte forma:

<app-nomeComponente [userData]="userData"></app-nomeComponente>

Por fim, no arquivo HTML do componente filho, utilizamos o dado da seguinte forma:

<p>Email: {{ userData.email }}</p>

<p>Função: {{ userData.role }}</p>
