Linux

Manipular arquivos

touch arquivo.txt Cria um arquivo txt

echo "ola teste" >> teste.txt Se o arquivo teste.txt não existir, ele é criado e é adicionado o texto nele. Se ele já existir o texto é incluído no arquivo. OBS: sem acentos

rm arquivo Apaga um arquivo

rm -rf pasta Apaga uma pasta e seu conteúdo

mv nome_arquivo destino_arquivo Move um arquivo para outro lugar

nano arquivo.txt Abre o arquivo no editor de texto "nano". Se o arquivo não existir ele será criado após salvar.

Comandos diversos

pwd Mostra todo o caminho da pasta atual

ls Mostra todos os arquivos da pasta atual

ls -a Mostra todos os arquivos da pasta atual incluindo ocultos

ls -la Mostra todos os arquivos da pasta atual com detalhes

cd caminho_da_pasta entra em uma pasta

cd .. Volta para o nível anterior da pasta

cd ../.. Volta dois níveis anteriores da pasta
