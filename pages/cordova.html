Site

Criar projeto

cordova create NomeDoApp

Definir com qual plataforma será compativel

cordova platform add android

cordova platform add browser

cordova platform add ios

Rodar o aplicativo

cordova run android

cordova run browser

cordova run ios

Para rodar no celular, precisa instalar o celular no android studio e depois conectar o celular

Para finalizar o app, aperta Ctrl+C no terminal

Se der algum erro desconhecido

cordova platform rm android (remove a plataforma android)

cordova platform add android (adiciona novamente a plataforma android)

Alterar nome da aplicação

Vai no arquivo config.xml e altera na segunda linha id="io.cordova.hellocordova" o nome hellocordova é o nome interno do sistema que deve mudar para um nome único, e na terceira linha, na tag name, é o nome que aparece na tela
