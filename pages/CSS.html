CSS

Aplicar propriedade a todo o documento

Exemplo:
* { box-sizing: border-box;
    margin: 0;
    padding: 0;}

Atributos para texto

Centralizar texto: text-align: center

Dar espaço no início da linha: text-indent: 40px;

Primeira letra maiúscula: text-transform: capitalize;

Todas maiúsculas: text-transform: uppercase;

Todas minúsculas: text-transform: lowercase;

Cor do texto: color: red;
