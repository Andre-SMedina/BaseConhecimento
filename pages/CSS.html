Atributos para texto

Centralizar texto: text-align: center

Dar espaço no início da linha: text-indent: 40px;

Primeira letra maiúscula: text-transform: capitalize;

Todas maiúsculas: text-transform: uppercase;

Todas minúsculas: text-transform: lowercase;

Cor do texto: color: red;

Ocultar e mostrar item

Coloca todo o conteudo que deve sumir, dentro de uma div com a classe "conteudo"

.conteudo {
            overflow: hidden;
            transition: 1s ease-out;
            max-height: 0px;
          }

Cria uma div com a classe item que contenha dentro a div com a classe conteudo.

.item:hover .conteudo {
            max-height: 2000px;
            transition: 2s ease-in;
          }

Assim, quando passar o mouse em cima da div item, a div conteudo será mostrada
