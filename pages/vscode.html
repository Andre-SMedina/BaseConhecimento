Inserir bloco de código personalizado (snippets)

  1. Abrir o "Command Palette" com o comando Ctrl+Shift+p
  2. digita snippets
  3. seleciona "snippets: configure user snippets"
  4. Escolha o tipo de arquivo que deseja aplicar a regra
  5. Remova o comentário que está abaixo de Example:
"Print to console": {
    "prefix": "x",
    "body": [
        "console.log('$1');",
        "$2"
    ],
    "description": "Log output to console"
            }

Na primeira linha você dá o nome que quiser para identificar o bloco. Em "prefix" coloca a sequência de letras que serão digitadas para inserir o bloco. Em "body", cada elemento no array representa uma linha que será inserida. "$1" indica onde vai começar o cursor, e na sequêcia vem "$2" que indica o segundo lugar onde vai ficar o cursor. "$0" indica onde vai ficar a última posição do cursor. Para dar um "tab", coloca "\t"
